# /efficiency — 工程效能

工程效能由主 Agent 直接处理。

## 执行流程

1. 明确效能提升主题和目标项目
2. 确认项目在 workspace 下是否有对应目录，如无则创建
3. 分析当前研发流程和效能瓶颈
4. 输出优化建议和效能指标
5. 输出到 `workspace/{项目名}/05-efficiency/`

## 支持的操作

- **metrics** — 效能指标定义
- **process** — 研发流程优化
- **cicd** — CI/CD 方案
- **review** — 代码审查流程

## 触发条件

当用户提到研发效能、CI/CD、开发流程、代码审查时自动使用。
