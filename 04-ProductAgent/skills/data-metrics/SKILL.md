# 数据指标 — data-metrics

## B 端 SaaS 核心指标体系

### 北极星指标

每个产品应有一个北极星指标，反映产品为用户创造的核心价值。

| 产品类型 | 北极星指标 |
|---|---|
| 协作工具 | 周活跃团队数 |
| CRM | 跟进的商机数 |
| ERP | 处理的订单数 |
| OA | 完成的审批数 |

### 指标层级

```
北极星指标
├── 获客 — 线索量、转化率、CAC
├── 激活 — 注册完成率、首次关键操作
├── 留存 — DAU/MAU、功能使用率、流失率
├── 收入 — MRR、ARR、ARPU、NDR
└── 口碑 — NPS、转介绍率
```

## 分析方法

### 漏斗分析

追踪用户转化路径，定位流失环节。

关键原则：
- 漏斗步骤不超过 7 步
- 每步有明确的数据口径
- 分维度对比（渠道、设备、时间）

### 留存分析

| 分析维度 | 说明 |
|---|---|
| 同期群 | 按注册月份分组的留存曲线 |
| 行为留存 | 使用特定功能后的留存变化 |
| 特征留存 | 不同用户属性的留存差异 |

### 分群分析

- **RFM** — Recency（最近活跃）、Frequency（频次）、Monetary（价值）
- **生命周期** — 新客、活跃、沉默、流失
- **价值分层** — 高/中/低价值客户

### A/B 测试

| 要素 | 说明 |
|---|---|
| 假设 | 明确的因果关系 |
| 样本量 | 达到统计功效 |
| 实验周期 | 覆盖完整周期（至少 1 周） |
| 显著性 | p < 0.05 |
| 实际显著性 | 效应量有业务意义 |

## 增长模型

### AARRR 漏斗

| 阶段 | 指标 | 策略 |
|---|---|---|
| Acquisition | 流量、线索 | 内容营销、SEO、合作伙伴 |
| Activation | 注册转化、首次使用 | 引导流程、模板预设 |
| Retention | 留存率、活跃率 | 持续价值、功能迭代 |
| Revenue | MRR、ARPU | 增购、升级、定价优化 |
| Referral | 推荐率、NPS | 客户案例、推荐奖励 |

### 增长公式

```
收入 = 流量 × 转化率 × ARPU
增长 = 新用户获取 + 老用户增购 - 流失
```
