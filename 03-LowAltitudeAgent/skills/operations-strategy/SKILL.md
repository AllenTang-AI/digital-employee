# 运营策略 — operations-strategy

## 运营模式

### 政府主导模式

- **特点** — 政府投资、公共服务、非营利
- **收入** — 财政拨款、政府购买服务
- **适用** — 监管平台、公共服务平台

### 政企合作模式

- **特点** — 政府 + 企业联合，共担风险
- **收入** — 服务费 + 政府补贴
- **适用** — 低空物流、低空旅游、城市巡查

### 企业主导模式

- **特点** — 市场化运营，追求盈利
- **收入** — 服务费、增值服务费
- **适用** — 商业平台、第三方服务平台

## 数据指标体系

### 北极星指标

> 日飞行架次

反映平台的核心价值：连接飞行器和空域，促成安全飞行。

### 指标层级

```
日飞行架次（北极星）
├── 用户 — 注册用户、活跃用户、企业用户
├── 飞行 — 飞行架次、飞行时长、航线数
├── 空域 — 开放空域、空域使用率
├── 安全 — 安全事故、违规事件
├── 服务 — 审批时效、告警响应
└── 收入 — 服务费、增值收入
```

### 核心指标定义

| 指标 | 定义 | 计算方式 |
|---|---|---|
| 日飞行架次 | 当日完成的飞行次数 | COUNT(flight WHERE date=today) |
| 活跃飞行器 | 当日有飞行记录的飞行器数 | COUNT(DISTINCT aircraft WHERE date=today) |
| 空域使用率 | 已使用空域面积 / 总开放面积 | used_area / total_area |
| 审批时效 | 从申报到审批的平均时长 | AVG(approval_time) |
| 安全事故率 | 安全事故数 / 总飞行架次 | accidents / flights |

## 增长策略

### 冷启动（0-1）

- **策略** — 政府背书 + 试点先行
- **目标** — 获取种子用户，跑通流程
- **关键** — 选择一个场景深度做透

### 早期增长（1-10）

- **策略** — 免费试用 + 标杆案例
- **目标** — 扩大用户规模
- **关键** — 打造可复制的成功案例

### 规模增长（10-100）

- **策略** — 增值服务 + 生态合作
- **目标** — 提升收入
- **关键** — 建立平台生态

## 商业化路径

| 阶段 | 收入来源 | 目标占比 |
|---|---|---|
| 建设期 | 政府投资 | 100% |
| 运营初期 | 政府补贴 + 基础服务费 | 70% + 30% |
| 成长期 | 服务费 + 增值服务费 | 60% + 40% |
| 成熟期 | 服务费 + 增值服务 + 数据服务 | 50% + 30% + 20% |
