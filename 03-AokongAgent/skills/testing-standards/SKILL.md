# 无人机测试标准 — testing-standards

## 测试类别与优先级

### P0 — Safety Critical (Must Pass)
| 测试项 | 目标 | 通过标准 |
|---|---|---|
| 电子围栏 | 超出限制区域自动悬停/返航 | 100% 触发 |
| 低电量返航 | 电量低于阈值自动 RTH | RTH 成功，安全着陆 |
| 失联保护 | 信号丢失后自动返航 | 30s 内触发 RTH |
| 紧急停机 | 手动触发停机 | 3s 内停机，无危险动作 |
| 避障 | 前方障碍物自动避让 | 2m 距离内 100% 避让 |

### P1 — Core Functions
| 测试项 | 目标 | 通过标准 |
|---|---|---|
| 起降 | 自动起飞/降落 | 5 次全部成功 |
| 悬停 | 定点悬停精度 | ±0.5m (GPS) / ±0.3m (RTK) |
| 航线飞行 | 按预设航点飞行 | 偏差 < 2m |
| 云台控制 | 角度调整、拍照/录像 | 响应 < 500ms |
| 通信距离 | 最大控制距离 | 达到标称值 90% |

### P2 — Nice to Have
| 测试项 | 目标 |
|---|---|
| 智能跟随 | 目标识别和跟随 |
| 手势控制 | 手势拍照/录像 |
| 自动环绕 | 围绕目标自动环绕拍摄 |

## 缺陷分类

| 级别 | 定义 | 示例 | 处理 |
|---|---|---|---|
| **Critical** | 安全隐患，可能导致坠机、伤人 | 返航失败、电池过热 | 立即修复，回归测试 |
| **Major** | 核心功能不可用 | 无法起飞、通信断连 | 本版本修复 |
| **Minor** | 功能可用但有瑕疵 | 云台抖动、界面卡顿 | 排入迭代计划 |
| **Cosmetic** | 不影响功能 | UI 文字对齐 | 有空再修 |

## 测试报告模板

- 产品/版本信息
- 测试环境（硬件、固件、环境条件）
- 测试执行摘要（总数/通过/失败/阻塞）
- 失败用例详情
- 缺陷汇总
- 风险评估和发布建议（Go / No-Go / Conditional）
